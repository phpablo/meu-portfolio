# Portfólio — Pablo Henrique Araújo

Portfólio totalmente redesenhado com uma estética tecnologia + Harry Potter, moderno e dinâmico. Foco em clean code, performance, acessibilidade e boas práticas de UI/UX.

## Visão Geral
- Hero com partículas voando e trilha mágica acompanhando o mouse.
- Layout minimalista, tipografia Orbitron + Cinzel.
- Seções: Início, Perfil, Feitiços de Código e Projetos.
- Responsivo, acessível e tema claro/escuro.

## Destaques
- Interações: tilt nos cards, efeitos de feitiço, contornos e sombras sutis.
- Semântica, contraste e foco visível.
- Metadados para SEO.

## Tecnologias
- `HTML5` semântico
- `CSS3` com variáveis e grids
- `JavaScript` (IntersectionObserver, interações, partículas)
- `Google Fonts` (Orbitron, Cinzel)
- `GitHub Pages`

## Executar localmente
1. Baixe ou clone este repositório.
2. Abra `index.html` no navegador ou coloque a pasta em um servidor local.

## Deploy no GitHub Pages
Opção A — Página do usuário:
- Crie o repositório `phpablo.github.io` no GitHub.
- Suba os arquivos (`index.html`, `styles.css`, `script.js`, `README.md`).
- Acesse `https://phpablo.github.io/` após a publicação.

Opção B — Página de projeto:
- Em `Settings` → `Pages`, selecione `Branch: main` e `Root`.
- Aguarde a publicação e acesse a URL indicada.

## Estrutura
- `index.html`: página principal e seções do site
- `styles.css`: tema escuro/claro, tipografia, grid e responsividade
- `script.js`: alternância de tema, animações e utilidades
